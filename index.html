Mesocast

Backend Engineer / System Architect
Weather data from radars to clients in under 10 seconds.

Mesocast is a weather forecasting application built with weather enthusiasts, meteorologists, and scientists in mind. It displays NEXRAD Level 3 and NEXRAD Level 2 weather radar products in real time. Mesocast also includes various other weather products such as satellite imagery, lightning data, and more. The main backbone of the application is its proprietary binary parser written in Golang that parses the raw radar data and delivers it to the client to be rendered in real time using OpenGL.
