BrewBot

Android Emulator Automation Software

BrewBot was my first major brainchild. The idea was to make a program that could automate Android emulators, such as BlueStacks and Nox, using image detection. Similar to the functionality of MyBot, but with the ability to work with more than just one Android application. I also used this as an opportunity to get more comfortable with C/C++ and the Win32 API. Unfortunately, I have let the application become abandon ware that I will return to sometime soon™.

Originally, BrewBot used the same image detection libraries as MyBot, but due to shortcomings of that library I started moving BrewBot to OpenCV. This move allowed for more complex operations; tracking object movement in a video or through a series of images being one of the more important ones. I also implemented LUA script support so users can easily make their own scripts for other apps.
